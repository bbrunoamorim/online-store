# Online store

This project is a simple front end for an online store

## Developed with

- JavaScript
- React
- Tailwind CSS

## How to install

```
# Clone this repo
  git@github.com:bbrunoamorim/online-store.git

# Install dependencies
  npm i

# Run the app
  npm start

# You're welcome to see it running here:
# https://online-store-eight-lake.vercel.app/
```
